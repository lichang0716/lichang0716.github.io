Cocoapods 安装梳理

发表于 2018-03-01   |   分类于 iOS   |  

RVM

1.安装 RVM（Ruby Version Manager)

curl -sSL https://get.rvm.io | bash -s stable
source ~/.bashrc
source ~/.bash_profile

2.获取可以安装的版本列表rvm list known

3.安装版本rvm install 2.4.1

4.设置版本rvm use 2.4.1 --default

5.安装 Ruby 之前，可能会遇到问题

Updating system.........
Installing required packages: autoconf, automake, libtool, pkg-config, coreutils, libyaml, readline, libksba, openssl@1.1

失败的话，使用 brew install 缺什么安装什么，安装完成之后，再安装 Ruby。

Ruby

1.查看 Ruby 版本 ruby -v

2.sudo gem update --system

3.查看当前源 gem sources -l

4.去掉源 gem source --remove

5.修改为 ruby-china 的源 gem sources -a https://gems.ruby-china.org/

6.安装 rails gem install rails

Cocoapods

Install

1.安装 Cocoapods sudo gem install cocoapods

2.第一次使用需要进行 pod setup

3.也可以进入 cd ~/.cocoapods/repo

4.使用命令 git clone https://github.com/CocoaPods/Specs.git

5.如果速度太慢的话参考下个模块，clone 结束后，移到 master 文件夹内

6.查看源 pod repo

7.查看 Pod 版本 pod --version

Unistall

1.移除 Pod 组件 which pod，rm -rf 对应路径的文件

2.移除 Gems 中的 Cocoapods 程序包 gem list 获取列表，然后使用 sudo gem uninstall
